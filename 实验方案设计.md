## 一、能不能用？——诚实的技术判断

### 1.1 有利因素

| 因素 | 证据 | 置信度 |
|------|------|:---:|
| 物理机制成立 | Helmholtz-Smoluchowski + 傅里叶导热 + Boussinesq 解，所有参数在已知物理范围内 | 🟢 高 |
| 电气安全 | 24V DC 属 SELV 范围，人体可直接接触无危险 | 🟢 高 |
| 排水能力有冗余 | 最不利工况下重力排水是需求的 28 倍 | 🟢 高 |
| 热安全有冗余 | 最保守模型下 ΔT 仅 2.3°C，距限值 50°C 有 12.7°C 裕度 | 🟢 高 |
| 有类似技术先例 | 英国 EKG 已有 25 年商业历史，BS 8006 已纳入 | 🟢 高 |
| 结构兼容 | 10mm 厚度可直接嵌入沥青摊铺流程 | 🟡 中 |

### 1.2 不确定因素（需要实验回答）

| 不确定因素 | 理论假设 | 潜在风险 | 影响程度 |
|-----------|----------|----------|:---:|
| Zeta 电位在真实路面水中的实际值 | ζ=50mV | pH/离子强度可能使 ζ 降至 10-20mV | 🔴 高 |
| 碳纤维阳极电化学腐蚀速率 | 石墨烯涂层保护 | 6-12 个月内腐蚀断丝 | 🔴 高 |
| 3D 芯体长期蠕变 | 25% 残余导水率 | 低于 25% 则排水能力不足 | 🟡 中 |
| 土工布淤堵 | O90 < 80μm | 细颗粒穿透堵塞芯体流道 | 🟡 中 |
| 电渗导致土体化学变化 | 忽略 | 阳极附近酸化/阴极碱化 → 弱化 | 🟡 中 |
| EO 排水效率的尺寸效应 | 线性外推 | 大尺寸下效率非线性衰减 | 🟡 中 |
| 间歇供电时 EO 脱附的持续性 | 瞬时脱附 | 断电后水膜重新吸附（松驰时间未知） | 🟡 中 |

### 1.3 结论

> **理论上可行，但必须经过 5 个阶段的实验验证才能回答"能不能用"。**  
> 实验室阶段（阶段 1-3）可回答约 70% 的不确定性。  
> 足尺试验（阶段 4-5）是最终判定依据。

## 四、阶段二：组件级实验（4-6 个月，预算 ~30 万元）

### 4.1 目的
制作小尺寸格栅原型，在受控环境下测试完整组件的电-热-流-力耦合性能。

### 4.2 样机制作

```
规格: 600 × 300 mm (覆盖 2 个完整电极周期)
  - 3 根阳极 (碳纤维束, 间距 600mm=cathode夹在中间, 周期0.3m)
  - 2 根阴极
  - 3D 芯体夹层
  - 上/下土工布热合封装
制作数量: 6 块 (3 测试 + 2 备用 + 1 破坏性)
```

### 4.3 实验矩阵

| 编号 | 实验名称 | 方法 | 测量指标 | 关键性 |
|:---:|----------|------|----------|:---:|
| **C1** | **电渗排水全尺寸单元** | 样机夹于 5cm 水稳层 + 3cm 沥青层之间, 饱和注水, 24V 间歇供电, 24h | 累积排水量, 阴极/阳极区含水率分布, 电流-时间曲线 | 🔴 |
| C2 | EO vs 纯重力对照 | 同 C1, 但不通电 | 自然排水量 vs EO 增强排水量 | 🔴 |
| **C3** | **热成像温升** | 同 C1, 红外热像仪全程监测 | 格栅表面温度分布, 热点识别 | 🔴 |
| C4 | 层间剪切 | 300×300mm 复合试件, 法向 500kPa, 剪切速率 1mm/min | 剪切强度 τ_max, 残余强度, 破坏面位置 | 🔴 |
| C5 | 沥青压实兼容性 | 样机夹于热沥青 (160°C) 间, 模拟摊铺碾压 | 压实后芯体厚度保留率, 导水率变化, 电极导通性 | 🟡 |
| C6 | 冻融循环 | -20°C ↔ +20°C, 20 次循环 | 冻融后排水性能变化, 芯体完整性 | 🟡 |
| C7 | 极性反转再生 | 每 24h 反转极性, 运行 30 天 | 电极腐蚀对比 (vs 单极性), 排水效率变化 | 🟡 |

### 4.4 阶段二通过标准

- [ ] 24h 排水量 ≥ 2.0 L/m²（允许实验室缩放损失）  
- [ ] EO 通电期间排水速率明显高于纯重力对照组（证明 EO 脱附有效）  
- [ ] 热成像：格栅表面无热点, 温升 ≤ 3°C（实验室环境）  
- [ ] 层间剪切：FS ≥ 1.5  
- [ ] 压实后芯体厚度 ≥ 5mm, 电极全部导通

## 六、阶段四：足尺加速加载试验（APT）（12-18 个月，预算 ~200-300 万元）

### 6.1 目的
在 1:1 足尺路面结构中，用加速加载设备模拟 10-15 年交通荷载，验证格栅在长期服役条件下的功能耐久性和结构安全性。

### 6.2 试验方案

```
试验设施: 交通运输部公路科学研究院 APT 试验场（北京/重庆）
           或长沙理工大学 NLAPT
           或山东交通学院 HVS

试验段设计:
├── 段 A: 传统结构（空白对照）          20m
├── 段 B: EODG 格栅（不通电，仅重力）   20m
├── 段 C: EODG 格栅（间歇供电）         20m
├── 段 D: EODG 格栅（连续供电，加速劣化）20m
└── 段 E: 传统排水层（对比方案）        20m

路面结构（统一）:
  沥青面层: 4cm AC-13 + 6cm AC-20 + 4cm AC-25 = 14cm
  EODG 格栅
  水稳基层: 20cm (水泥剂量 5%)
  级配碎石底基层: 20cm
  路基: 压实土, E0 ≥ 40MPa

加载参数:
  轴载: 100 kN (标准双轮组)
  轮压: 0.7 MPa
  速度: 8 km/h
  目标: 500 万次 (模拟 10 年重载交通)
```

### 6.3 监测方案

| 监测项目 | 传感器/方法 | 频率 |
|----------|------------|:---:|
| 层间含水率 | TDR 水分计 ×32（每段 8 个） | 实时 |
| 格栅温度 | 热电偶 ×20 | 实时 |
| 电流/电压/功率 | 供电系统数据采集 | 实时 |
| 路表弯沉 | FWD 落锤式弯沉仪 | 每 50 万次 |
| 层间滑移 | 水平位移计 | 连续 |
| 裂缝发展 | 人工巡检 + 激光扫描 | 每 10 万次 |
| 芯体厚度 | 钻芯取样 | 初始 + 250万次 + 终了 |
| 排水量 | 集水沟流量计 | 连续 |

### 6.4 阶段四通过标准

- [ ] 500 万次加载后格栅芯体厚度 ≥ 4mm（初始 8mm 的 50%）  
- [ ] 层间含水率：段 C（EO 供电）比段 A（对照）低 ≥ 30%  
- [ ] 段 C 路表弯沉增长率 ≤ 段 A 的 50%（含水控制延缓结构劣化）  
- [ ] 无层间滑移裂缝  
- [ ] 电极导通率 ≥ 90%（单点断丝可容忍）  
- [ ] 500 万次后排水能力不低于初始的 50%

## 十、最关键的一个实验

如果预算极度有限只能做一个实验，那就是：

> **阶段一 M4.1 + M4.4 合并**：在 10×10cm 压实水稳层试块上，24V 叉指电极，测量：(1) 1h 通电排水量；(2) 断电后 3h 含水率回弹量。

这两个数字会直接告诉我们 EO 脱附范式是否成立。如果断电后水膜在 3h 内回弹超过 50%，说明 EO 释放-重力排水的思路可能需要重新设计（比如需要更短的供电间隔或连续供电）。
